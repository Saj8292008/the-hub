## Part 4: Deal Scoring Recommendations

### Proposed Scoring Formula

```
Deal Score = Base Score + Category Modifiers + Timing Bonus + Condition Bonus

Base Score (0-100):
- (Market Value - Listed Price) / Market Value × 100
- Capped at 100 for extreme deals (may indicate scam)

Category Modifiers:
- Watches: +10 if from AD, +5 if box/papers, -10 if no service history
- Sneakers: +10 if DS, +5 if OG all, -15 if worn, -5 per flaw
- Cars: +10 if CPO, +5 if 1-owner, -10 per major accident

Timing Bonus:
- +5 if posted within 24h
- +3 if posted within 48h
- 0 if older

Condition Bonus (from description analysis):
- "BNIB/DS/New": +5
- "Like new/Excellent": +3
- "Good/Used": 0
- "Fair/Beater": -5
```

## Part 6: Red Flags & Scam Indicators

### Price-Based Red Flags

- **Watches:** >30% below market = likely fake or stolen
- **Sneakers:** >40% below StockX = likely fake
- **Cars:** >25% below KBB = salvage/flood/odometer issue

### Description Red Flags

**Watches:**
- "No papers" on recent Rolex (should have warranty card)
- Mismatched serial numbers
- "Overseas" seller with US-priced watch

**Sneakers:**
- No tagged photos of size tag
- "Receipt available" but no photos of it
- Multiple sizes available (bulk fake operation)

**Cars:**
- "Rebuilt title" buried in description
- "Previous rental car" (high wear)
- Inconsistent odometer in photos vs listing
