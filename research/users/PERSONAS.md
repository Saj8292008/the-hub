## Design Implications

### For Resellers:
- Dashboard-first experience
- Quick-add inventory flows
- Profit always visible
- Bulk actions everywhere

### For Collectors:
- Wishlist is primary view
- Visual price history
- Notification settings prominent
- Community trust signals

### For Bargain Hunters:
- Simple, clean interface
- One-tap actions
- "Deal or No Deal" obvious
- Digest/summary views

### For Dealers:
- Power user shortcuts
- Export everything
- API documentation
- SLA and support tiers
