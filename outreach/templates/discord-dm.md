# Discord DM Templates

## Finding Prospects on Discord

**Servers to join:**
- Watch trading servers (search Discord server lists)
- Sneaker cook groups / resell communities
- r/Watchexchange and r/sneakermarket Discord servers
- Brand-specific groups (Rolex, Jordan, Yeezy communities)

**How to find sellers:**
- Look in #for-sale or #wts channels
- Note frequent posters with multiple listings
- Check who has "Verified Seller" roles
- Active in #price-check channels = knows the market

## Discord Best Practices

1. **Be active in servers first** - Don't join and immediately DM
2. **Establish presence** - Help people, answer questions, be known
3. **Check if DMs are open** - Some users block server DMs
4. **No mass DMs** - Discord bans for spam quickly
5. **Mutual server reference** - "Saw you in [server]" builds trust
6. **Don't poach openly** - Admins will ban you for soliciting members
7. **Offer value first** - Share useful info before asking for anything
8. **Voice chat if possible** - Higher conversion, builds real connection
