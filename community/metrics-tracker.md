## Key Performance Indicators (KPIs)

### Primary Metrics

| Metric | Definition | Target | Frequency |
|--------|------------|--------|-----------|
| **Total Members** | Sum across all platforms | +20%/month | Weekly |
| **Daily Active Users** | Unique engaged members/day | 10% of total | Daily |
| **DAU/MAU Ratio** | Daily actives / Monthly actives | >20% | Monthly |
| **Premium Conversion** | Free â†’ Paid conversion rate | 3-5% | Monthly |
| **Churn Rate** | Members leaving / Total members | <5%/month | Monthly |

### Engagement Metrics

| Metric | Definition | Target | Frequency |
|--------|------------|--------|-----------|
| **Messages/Day** | Community messages per day | 100+ | Daily |
| **Posts Engagement** | Reactions + replies per post | 10+ | Per post |
| **Poll Participation** | Votes / Post views | 20%+ | Per poll |
| **User-Generated Content** | Member-shared deals/day | 5+ | Daily |
| **Questions Answered** | % questions getting responses | 95%+ | Weekly |

### Growth Metrics

| Metric | Definition | Target | Frequency |
|--------|------------|--------|-----------|
| **New Members/Week** | New joins across platforms | +100 | Weekly |
| **Referral Rate** | New members from referrals | 20%+ | Monthly |
| **Newsletter Signups** | New email subscribers/week | +50 | Weekly |
| **Organic Traffic** | Non-paid website visitors | +10%/month | Monthly |

### Quality Metrics

| Metric | Definition | Target | Frequency |
|--------|------------|--------|-----------|
| **Response Time** | Avg time to first response | <30 min | Weekly |
| **Support Satisfaction** | Post-support rating | 4.5/5 | Monthly |
| **NPS Score** | Net Promoter Score | >50 | Quarterly |
| **Spam/Violation Rate** | Bad content / Total content | <1% | Weekly |

## Data Collection Checklist

### Daily (5 min)
- [ ] Log platform member counts
- [ ] Note any significant events

### Weekly (15 min)
- [ ] Pull engagement metrics
- [ ] Calculate week-over-week changes
- [ ] Compile top content
- [ ] Review support trends
- [ ] Draft weekly report

- [ ] Compile all weekly data
