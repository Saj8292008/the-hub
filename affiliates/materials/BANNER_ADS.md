## üìÅ File Delivery Specs

### Static Banners
- Format: PNG, JPG
- Max file size: 150KB
- Resolution: 72 DPI

### Animated Banners
- Format: GIF or HTML5
- Max file size: 200KB
- Animation length: 15 seconds max
- Must end on clear CTA frame
- Looping: 3x then stop

### Retina Versions
- Provide @2x versions for all sizes
- Same file size limits
